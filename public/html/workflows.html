CSV File Controlled Field Assessor

Creates reports for Darwin Core fields that are supposed to have controlled values. Reports include counts of distinct values and recommendations for those that can be found in the vocabulary files.

CSV File Aggregator

Create a file that combines two separate input files with the distinct columns and all records from both.

CSV File Geography Cleaner

Creates reports for Darwin Core higher geography fields. Reports include counts of distinct values and recommendations for those that
can be found in the vocabulary files. Also creates an output file that replaces values of Darwin Core geography fields with standardized
values and adds fields to include the original field values.

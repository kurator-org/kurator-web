Kurator Web is a series of open source data curation tools that will run a given set of biodiversity data against verified databases such as Catalogue of Life, Global Biodiversity Information Facility (GBIF), Global Names Index (GNI) and others (see full list here). Kurator tools data quality tools include: taxonomic name resolution, georeferencing, and duplicate specimen detection.

Checking your data against these databases using the Kurator tools will provide you with a “data quality” report showing you possible errors that you can then correct = better data quality to help facilitate digitization and biodiversity research!

Authorities that Kurator checks against

  • Catalogue of Life (CoL) www.catalogueoflife.org
  • GEOLocate www.museum.tulane.edu/geolocate
  • Global Biodiversity Information Facility (GBIF) www.gbif.org
  • Global Names Index  (GNI) http://gni.globalnames.org
  • Flora of North America (FNA) www.floranorthamerica.org
  • Index Fungorum (IF) www.indexfungorum.org
  • International Plants Names Index (IPNI) www.ipni.org
  • World Register of Marine Species (WoRMS) www.marinespecies.org

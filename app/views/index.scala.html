Kurator tools and validators can help you check, standardize and clean biodiversity data before doing statistical analysis or research. Kurator Web tools detect and flag data quality issues and propose amendments while keeping track of provenance.



Login or Register a user account to access the web application and run the workflows or see the About page for more info.



How Kurator Works

  • Checks data sets for internal consistency and validation
  • Checks data sets against external authority resources
  • Identifies potential problems or errors in the data and produces data quality reports
